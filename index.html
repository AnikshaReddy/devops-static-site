About the Project

This project demonstrates how to host a static website using Google Cloud Storage. It applies DevOps principles to automate deployment and improve website availability.

Objectives

  • Deploy a static website on Google Cloud Platform
  • Use Google Cloud Storage for hosting files
  • Improve scalability and availability
  • Enable easy maintenance and updates
  • Implement basic DevOps practices

Features

Cloud Hosting

Host static files using GCP cloud infrastructure.

Scalable

Handles increased traffic with reliable performance.

Cost Effective

Uses pay-as-you-go cloud services for reduced cost.
